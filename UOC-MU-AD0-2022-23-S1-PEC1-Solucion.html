1 Introduccin

La solucin de la PEC tiene dos objetivos principales: - Por un lado, debe mostrar un modelo de resolucin de las preguntas planteadas que sirva de ejemplo de como se poda haber resuelto las preguntas y de comparacin con vuestra propia solucin. - Adems debe contener explicaciones sobre el desarrollo de la PEC, que, normalmente no formaran parte de un informe, como es el caso de esta introduccin. Tal como se os peda, el cdigo no aparecer dentro del texto, aunque con fines didcticos lo podris mostrar cuando as lo deseis. Esto se puede hacer con un simple comando de Rmarkdown

1.1 Objetivos

El objetivo principal de este trabajo es realizar un anlisis exploratorio de unos datos de microarrays, descargados de la base de datos Gene Expression Omnibus (GEO) utilizando el programa estadstico R y las libreras para anlisis de datos micos integradas en Bioconductor. En concreto se requiere que los datos descargados se almacenen en un objeto de clase ExpressionSet y que se acceda a ellos a travs de este objeto.

2 Materiales y Mtodos

Los datos con los que se trabajar se descargarn de GEO utilizando el paquete GEOquery de Bioconductor. Este paquete permite descargar los datos indicados (por un identificador GSE o GDS) y crear con ellos una estructura de datos del tipo expressionSet que contiene una matriz de datos preprocesados (habitualmente normalizados, as como una tabla con informacin sobre covariables y tras aspectos del experimento.

La exploracin se llevar a cabo siguiendo la plantilla de un caso de estudio, Analisis_de_datos_omicos-Ejemplo_0-Microarrays y por lo tanto est relativamente bien pautada. Bsicamente dicha exploracin consistir en: - Anlisis univariante de los datos, mediante boxplots y/o histogramas para estudiar la forma general de los mismos. - Anlisis multivariante de los datos, mediante Anlisis de Componentes Principales y Agrupamiento Jerrquico, para determinar si los grupos que aparezcan (en caso de hacerlo) parecen relacionarse con las fuentes de variabilidad del estudio o, si por el contrario, podran haber otras fuentes de variabilidad como efectos batch.

Las tcnicas mencionadas son muy habituales en estadstica y bioinformtica por lo que no nos entretendremos en explicarlas. Pueden verse resumidas en los materiales de la asignatura (documento Mdulo 1. Preliminares).

3.2.1 Exploracin multivariante

Un anlisis en componentes principales puede facilitar la visualizacin de los datos en dimensin reducida y, sobretodo, detectar posibles patrones que no se detecten a simple vista.

El PCA transforma las variables originales de forma que las nuevas componentes (las variables transformadas) resultan tener dos propiedades muy interesantes: - Son independientes entre ellas, es decir explican propiedades distintas de los datos). - Cada componente explica un porcentaje de variabilidad mayor que la anterior, con lo que suele bastar con las dos o tres primeras componentes para obtener una visualizacin de los datos en dimensin reducida.

En primer lugar se realiza el clculo de las componentes principales:

resPC <- prcomp(t(exprs(eSet)))

3.3 Discusin

El anlisis exploratorio, realizado tras tomar logaritmo sobre los datos, ha puesto de manifiesto que existen dos fuentes de variacin distintas. La mayor, relacionada con el origen de las muestras, no estaba especificada en el diseo del estudio. El efecto del tratamiento no aparece como la principal fuente de variabilidad lo que puede, probablemente, atribuirse a la confusin entre ambas fuentes.

Aparte de este problema inesperado, los datos no presentan evidencias de problemas: las distribuciones de las muestras son similares y no hay valores faltantes o cero, lo que se habra evidenciado al tomar logaritmos, por lo que en una situacin real se procedera a investigar si es posible eliminar lo que aparenta ser un efecto batch y llevar a cabo el anlisis.
